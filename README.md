## O que é o move.it?
Move.it é um projeto desenvolvido durante a Next Level Week #4 da Rocketseat com finalidade de propor desafios tanto físicos quanto oculares para usuários que passam muito tempo no computador.

## Tecnologias
- ReactJS
- NextJS
- Typescript
